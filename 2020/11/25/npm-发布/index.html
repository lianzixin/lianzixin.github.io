小工具 nrm

npm 包有很多镜像源，有时候需要切换 npm 的源，而 nrm 包就是解决快速切换问题的
nrm 可以在不同的 npm 原地址之间轻松快速地切换。

安装 nrm

运行命令：npm install -g nrm

列出可选择的源

运行命令：nrm ls

前面带 * 号的表示正在使用的源

切换使用的源

运行命令：nrm use npm
将切换到官方镜像源

添加源

运行命令：nrm add <registry-name> <url>
添加命名为 registry-name，地址为 url 的源

删除源

运行命令：nrm del <registry-name>
删除命名为 registry-name 的源

错误码 403 Forbidden，Private mode enable, only admin can publish this module

有可能 npm 源设定错误，切换为官方源

错误码 402 Payment Required ，You must sign up for private packages

接入等级为 private，需要付费，切换为公共

定义自己的 npm 包信息

运行命令：npm init --scope=@name
使用包范围为 @name，该范围需要自己在 npm 官网事先定义，然后依次填其他信息

发布自己的 npm 包

运行命令：npm publish
发布成功将会看到下面信息
