how-to-use-style-in-react

2020-12-02 how to

安装依赖

  • styled-components：npm install --save styled-components

CSS 优先级

优先级关系：内联 > ID选择器 > 类选择器 > 标签选择器
优先级关系可简单化为数组：(内联次数，ID 次数，类次数，标签次数)
比较方式为从左到右依次比较

ID 选择器
声明写法为：#xxx {}
使用写法为：<p id="xxx">

类选择器
声明写法为：.xxx {}
使用写法为：<p className="xxx">

标签选择器
声明写法为：xxx {}
使用写法为：<p>

例子实践

从左往右依次做比较，上述排列顺序按照递增进行排序

使用模板

创建特定 style 的特定元素，使用 . 函数

具体使用，同元素正常使用
